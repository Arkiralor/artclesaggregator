# News Article Aggregator
A webscraper to aggregate news articles from a single or set of websites.

## Pre-requisites:
  ### Selenium
  ### Webdriver
  ### Pandas 
    (or any other datastructure framework)

## How to run:
  1. Change chromedriver specs as per your installation.

  2. Change tag-names and class-names in the scraper function as per the CSS layout of the targetted website.

## Output:
  Stored as a .CSV file appended with the execution date in the root directory.
